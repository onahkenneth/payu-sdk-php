Create Payment Real-Time Recurring

This sample code demonstrates how you can process a direct Real-Time Recurring credit card transaction.

/** @var CreditCard $card */ $card = require __DIR__ . '/../../safestore/setup-real-time-recurring-credit-card.php'; $reference = $card->getReturn()->getMerchantReference(); $pmId = $card->getReturn()->getPaymentMethodsUsed()->getId(); $userId = $card->getCustomer()->getCustomerInfo()->getCustomerId(); use PayU\Api\Amount; use PayU\Api\CreditCardToken; use PayU\Api\FundingInstrument; use PayU\Api\Customer; use PayU\Api\CustomerInfo; use PayU\Api\Payment; use PayU\Api\Transaction; use PayU\Soap\ApiContext; use PayU\Api\PaymentMethod;

Credit card token

Saved payment method id from a previous call to setup-real-time-recurring-credit-card.php

$creditCardToken = new CreditCardToken(); $creditCardToken->setCreditCardId($pmId) ->setCvv2('123');

FundingInstrument

A resource representing a Customer's funding instrument. For stored credit card payments, set the CreditCardToken field on this object.

$fi = new FundingInstrument(); $fi->setCreditCardToken($creditCardToken);

CustomerInfo

A resource representing a CustomerInfo with details of the customer.

$ci = new CustomerInfo(); $ci->setCustomerId($userId);

Customer

A resource representing a Customer that funds a payment.

$customer = new Customer(); $customer->setPaymentMethod(PaymentMethod::TYPE_REAL_TIME_RECURRING) ->setCustomerInfo($ci) ->setFundingInstrument($fi);

Amount

Lets you specify a payment amount. You can also specify additional details such as shipping, tax.

$amount = new Amount(); $amount->setCurrency("ZAR") ->setTotal(100.00);

Transaction

A transaction defines the contract of a payment - what is the payment for and who is fulfilling it.

$transaction = new Transaction(); $transaction->setAmount($amount) ->setDescription("Payment description") ->setInvoiceNumber($reference);

Payment

A Payment Resource, create one with intent set to 'payment'

$payment = new Payment(); $payment->setIntent(Transaction::TYPE_PAYMENT) ->setCustomer($customer) ->setTransaction($transaction);

Enterprise API integration

$apiContext[0]->setAccountId("acct1") ->setIntegration(ApiContext::ENTERPRISE);

$request = clone $payment;
